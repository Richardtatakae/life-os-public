# Tauri icons

Build agent: generate these before first `cargo tauri build`. Use `npx @tauri-apps/cli@latest icon path/to/source.png` from a 1024x1024 source PNG.

Required files:
- `32x32.png`
- `128x128.png`
- `128x128@2x.png`
- `icon.icns` (macOS)
- `icon.ico` (Windows)

Source: a "Life OS" wordmark or a mountain-with-crown emblem in the dark-neon palette (`#0a0a1a` background, emerald `#10b981` foreground).
